# TXT Auto Reader（自动逐字阅读器）

一个纯前端的 TXT 文本自动阅读器，  
以 **“类似 AI 输出，一个字一个字吐出来”** 的方式展示文本，  
适合长文本阅读、专注阅读、慢速理解。

👉 在线体验（GitHub Pages）：  
https://123qweraz.github.io/autoreader/

---

## ✨ 功能特性

- 📄 **加载 TXT 文本**
- 🧠 **逐字输出**（模拟 AI 打字效果）
- ⏸️ **暂停 / 继续**
  - 按钮控制
  - `Space` 空格键快捷控制
- ⏩ **可调阅读速度**
- 📍 **按行阅读**
  - 可跳转指定行
  - 不会每次从头开始
- 📊 **阅读进度显示**
  - 总行数
  - 当前阅读行
- 👀 **阅读友好布局**
  - 当前行高亮
  - 上下文行辅助阅读
  - 垂直居中显示（阅读区不乱跑）
- 🔠 **字体 / 字号可调整**
